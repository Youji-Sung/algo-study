# 검증수

## 전지성

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<int> nums(5);
    int sum = 0;

    for(int& num : nums) {
        cin >> num;
        sum += num * num;
    }

    cout << sum % 10;

    return 0;
}
```
## 성유지

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    int res = 0;

    for (int i = 0; i < 5; ++i) {
        cin >> num;
        res += num*num;
    }
    
    cout << res % 10;
}
```
